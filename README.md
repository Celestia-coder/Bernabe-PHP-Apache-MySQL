# 📄 Student Management System (PHP + MySQL)

## 📌 Overview

This project is a simple web-based student management system that demonstrates basic CRUD (Create, Read, Update, Delete) operations using PHP, Apache, and MySQL. It allows users to manage student records efficiently through a clean web interface.

## ✨ Features

* **Add Student Records** – Insert new student information (e.g., name, email, course, etc.) into the database.

* **Display Student Records** – View all student records in a structured HTML table.

* **Update Student Information** – Edit and modify existing student details.

* **Delete Student Records** – Remove unwanted student entries from the database.

## 🎨 Styling

Basic styling is applied using inline CSS:

* Clean table layout with borders and hover effects
* Form styling for better readability
* Buttons styled for different actions (Add, Edit, Delete, Update)
* Responsive container and spacing


## 🛠 Tools Used

* PHP – Backend logic and server-side processing
* Apache – Web server to host and serve the application
* MySQL – Database to store and manage student records
* HTML/CSS – Frontend structure and styling
